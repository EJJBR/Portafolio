Portafolio personal diseñado y desarrollado desde cero para mostrar mis proyectos, habilidades y educación como futuro Ingeniero de Software.

## 🛠️ Tecnologías Utilizadas

Este proyecto fue construido sin el uso de librerías ni frameworks externos para mantener un código limpio, rápido y 100% personalizado:
- **HTML5** Semántico
- **CSS3** (Variables CSS, Flexbox, Grid, Animaciones clave)
- **JavaScript (Vanilla)** para interactividad y efectos sencillos

## 💻 Características
- **Totalmente Responsivo:** Se adapta perfectamente a dispositivos móviles y escritorio.
- **Diseño Personalizado:** Paleta de colores consistente basada en variables (Opera GX Theme).
- **Código Optimizado:** Estructura modular separando lógica (JS) y diseño (CSS) del contenido (HTML).
